# Superstore Python Analysis

## Objective

Analyze retail sales data using Python, Pandas, and Matplotlib to identify business trends and top-performing categories.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook / Google Colab

## Key Findings

* Technology generated the highest sales.
* West Region generated the highest profit.
* Sales increased over time.
* Canon imageCLASS 2200 Advanced Copier was the top-selling product.

## Skills Demonstrated

* Data Cleaning
* Data Analysis
* Data Visualization
* Pandas GroupBy Operations
* Business Intelligence
